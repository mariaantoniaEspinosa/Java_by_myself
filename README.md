# Exercícios Básicos em Java usando IntelliJ IDEA e JDK padrão
## Comandos e Conceitos usados ☕

*System.out.println("Texto");* = exibe mensagens e pula linha 

*System.out.print("Texto");* = exibe mensagens sem pular linha

*import java.util.Scanner;*

*Scanner sc = new Scanner(System.in);* = Lê os dados digitados pelo usuário

*int num = sc.nextInt();* = Leitura de um inteiro

*double valor = sc.nextDouble();* = Leitura de decimal

*String nome = sc.nextLine();* = Leitura de texto

### Tipos de dados 

*int numero;* = numeros inteiros

*double nota;* = numeros decimais

*String texto;* = texto
